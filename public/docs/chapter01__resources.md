# 1.4. Recursos

## 1.4.1. FAQ

Se durante a construção do sistema LFS você encontrar quaisquer erros, tiver quaisquer perguntas ou achar que há um erro de digitação no livro, por favor, comece consultando a lista de Perguntas Frequentes (FAQ), localizada em https://www.linuxfromscratch.org/faq/.

## 1.4.2. Listas de Discussão

O servidor linuxfromscratch.org hospeda várias listas de discussão usadas para o desenvolvimento do projeto LFS. Essas listas incluem as principais listas de desenvolvimento e suporte, entre outras. Se você não conseguir encontrar uma resposta para o seu problema na página de FAQ, o próximo passo seria pesquisar nas listas de discussão em https://www.linuxfromscratch.org/search.html.

Para informações sobre as diferentes listas, como se inscrever, locais dos arquivos e informações adicionais, visite https://www.linuxfromscratch.org/mail.html.

## 1.4.3. IRC

Vários membros da comunidade LFS oferecem assistência via Internet Relay Chat (IRC). Antes de usar este suporte, por favor, certifique-se de que sua pergunta já não foi respondida na FAQ do LFS ou nos arquivos das listas de discussão. Você pode encontrar a rede IRC em irc.libera.chat. O canal de suporte é chamado #lfs-support.

## 1.4.4. Sites Espelho

O projeto LFS possui vários espelhos mundiais para tornar o acesso ao site e o download dos packages necessários mais convenientes. Por favor, visite o site do LFS em https://www.linuxfromscratch.org/mirrors.html para uma lista dos espelhos atuais.

## 1.4.5. Informações de Contato

Por favor, direcione todas as suas perguntas e comentários para uma das listas de discussão do LFS (veja acima).
