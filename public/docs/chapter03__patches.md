# 3.3. Patches Necessários

Os patches também fazem pequenas modificações para tornar os packages mais fáceis de usar. Os seguintes patches serão necessários para construir um sistema LFS:

Além dos patches obrigatórios acima, existem vários patches opcionais criados pela comunidade LFS. Esses patches opcionais resolvem problemas menores ou habilitam funcionalidades que não são habilitadas por padrão. Sinta-se à vontade para consultar o banco de dados de patches localizado em https://www.linuxfromscratch.org/patches/downloads/ e adquirir quaisquer patches adicionais para atender às necessidades do seu sistema.
