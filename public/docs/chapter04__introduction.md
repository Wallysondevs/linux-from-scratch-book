# 4.1. Introdução

Neste capítulo, realizaremos algumas tarefas adicionais para preparar a construção do sistema temporário. Criaremos um conjunto de diretórios em $LFS (nos quais instalaremos as ferramentas temporárias), adicionaremos um usuário sem privilégios e criaremos um ambiente de build apropriado para esse usuário. Também explicaremos as unidades de tempo (“SBUs”) que usamos para medir o tempo necessário para construir pacotes LFS e forneceremos algumas informações sobre as suítes de teste de pacotes.
