# 5.1. Introdução

Este capítulo mostra como construir um cross-compiler e suas ferramentas associadas. Embora aqui a cross-compilação seja simulada, os princípios são os mesmos de uma cross-toolchain real.

Os programas compilados neste capítulo serão instalados no diretório $LFS/tools para mantê-los separados dos arquivos instalados nos capítulos seguintes. As bibliotecas, por outro lado, são instaladas em seu local final, já que pertencem ao sistema que queremos construir.
