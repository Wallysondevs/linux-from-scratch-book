# 7.11. Texinfo-7.2

O pacote Texinfo contém programas para ler, escrever e converter páginas info.

## 7.11.1. Instalação do Texinfo

Prepare o Texinfo para compilação:

```bash
./configure --prefix=/usr
```

Compile o pacote:

```bash
make
```

Instale o pacote:

```bash
make install
```

Detalhes sobre este pacote estão localizados em [Seção 8.72.2, “Conteúdo do Texinfo.”](#/page/chapter08__texinfo)
