# 8.39. Gperf-3.3

Gperf gera uma função hash perfeita a partir de um conjunto de chaves.

## 8.39.1. Instalação do Gperf

Prepare o Gperf para compilação:

```bash
./configure --prefix=/usr --docdir=/usr/share/doc/gperf-3.3
```

Compile o package:

```bash
make
```

Para testar os resultados, execute:

```bash
make check
```

Instale o package:

```bash
make install
```

## 8.39.2. Conteúdo do Gperf

### Breves Descrições
