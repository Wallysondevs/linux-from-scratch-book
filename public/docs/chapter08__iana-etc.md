# 8.4. Iana-Etc-20250807

O pacote Iana-Etc fornece dados para serviços de rede e protocolos.

## 8.4.1. Instalação do Iana-Etc

Para este pacote, nós só precisamos copiar os arquivos para o local:

```bash
cp services protocols /etc
```

## 8.4.2. Conteúdo do Iana-Etc

### Descrições Breves

/etc/protocols

Descreve os vários protocolos de Internet DARPA que estão disponíveis no subsistema TCP/IP

/etc/services

Fornece um mapeamento entre nomes textuais amigáveis para serviços de internet, e seus números de porta atribuídos subjacentes e tipos de protocolo
