# 8.74. MarkupSafe-3.0.2

MarkupSafe é um módulo Python que implementa uma string segura de marcação XML/HTML/XHTML.

## 8.74.1. Instalação do MarkupSafe

Compile o MarkupSafe com o seguinte comando:

```bash
pip3 wheel -w dist --no-cache-dir --no-build-isolation --no-deps $PWD
```

Este pacote não inclui uma suíte de testes.

Instale o pacote:

```bash
pip3 install --no-index --find-links dist Markupsafe
```
