# 8.44. XML::Parser-2.47

O módulo XML::Parser é uma interface Perl para o parser XML de James Clark, Expat.

## 8.44.1. Instalação do XML::Parser

Prepare o XML::Parser para compilação:

```bash
perl Makefile.PL
```

Compile o pacote:

```bash
make
```

Para testar os resultados, execute:

```bash
make test
```

Instale o pacote:

```bash
make install
```
