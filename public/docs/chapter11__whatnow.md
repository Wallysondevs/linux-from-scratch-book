# 11.4. Recursos Adicionais

Obrigado por ler este livro LFS. Esperamos que você tenha achado este livro útil e aprendido mais sobre o processo de criação do sistema.

Agora que o sistema LFS está instalado, você pode estar se perguntando “E agora?” Para responder a essa pergunta, compilamos uma lista de recursos para você.

- Manutenção Bugs e avisos de segurança são relatados regularmente para todo o software. Como um sistema LFS é compilado a partir do código-fonte, cabe a você manter-se atualizado sobre esses relatórios. Existem vários recursos online que rastreiam esses relatórios, alguns dos quais são mostrados abaixo: [Avisos de Segurança LFS](https://www.linuxfromscratch.org/lfs/advisories/) Esta é uma lista de vulnerabilidades de segurança descobertas no livro LFS após sua publicação. [Lista de E-mails de Segurança de Código Aberto](https://seclists.org/oss-sec/) Esta é uma lista de e-mails para discussão de falhas de segurança, conceitos e práticas na comunidade de Código Aberto.

- [Avisos de Segurança LFS](https://www.linuxfromscratch.org/lfs/advisories/) Esta é uma lista de vulnerabilidades de segurança descobertas no livro LFS após sua publicação.

- [Lista de E-mails de Segurança de Código Aberto](https://seclists.org/oss-sec/) Esta é uma lista de e-mails para discussão de falhas de segurança, conceitos e práticas na comunidade de Código Aberto.

- Dicas LFS As Dicas LFS são uma coleção de documentos educacionais enviados por voluntários da comunidade LFS. As dicas estão disponíveis em [https://www.linuxfromscratch.org/hints/downloads/files/](https://www.linuxfromscratch.org/hints/downloads/files/).

- Listas de e-mails Existem várias listas de e-mails LFS às quais você pode se inscrever se precisar de ajuda, quiser se manter atualizado com os últimos desenvolvimentos, quiser contribuir para o projeto e muito mais. Veja [Capítulo 1 - Listas de E-mails](#/page/chapter01__resources) para mais informações.

- O Projeto de Documentação Linux O objetivo do Projeto de Documentação Linux (TLDP) é colaborar em todas as questões da documentação Linux. O TLDP apresenta uma grande coleção de HOWTOs, guias e páginas de manual. Ele está localizado em [https://tldp.org/](https://tldp.org/).
