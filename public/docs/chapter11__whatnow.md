# 11.4. Recursos Adicionais

Obrigado por ler este livro LFS. Esperamos que você tenha achado este livro útil e que tenha aprendido mais sobre o processo de criação do sistema.

Agora que o sistema LFS está instalado, você pode estar se perguntando “E agora?” Para responder a essa pergunta, compilamos uma lista de recursos para você.

- Manutenção Bugs e avisos de segurança são relatados regularmente para todo o software. Como um sistema LFS é compilado a partir do código-fonte, cabe a você manter-se atualizado sobre esses relatórios. Existem vários recursos online que rastreiam esses relatórios, alguns dos quais são mostrados abaixo: LFS Security Advisories Esta é uma lista de vulnerabilidades de segurança descobertas no livro LFS após sua publicação. Open Source Security Mailing List Esta é uma lista de discussão para debate sobre falhas de segurança, conceitos e práticas na comunidade Open Source.

- LFS Security Advisories Esta é uma lista de vulnerabilidades de segurança descobertas no livro LFS após sua publicação.

- Open Source Security Mailing List Esta é uma lista de discussão para debate sobre falhas de segurança, conceitos e práticas na comunidade Open Source.

- LFS Hints Os LFS Hints são uma coleção de documentos educacionais enviados por voluntários da comunidade LFS. Os hints estão disponíveis em https://www.linuxfromscratch.org/hints/downloads/files/.

- Listas de e-mail Existem várias mailing lists LFS às quais você pode se inscrever se precisar de ajuda, quiser se manter atualizado com os desenvolvimentos mais recentes, quiser contribuir para o projeto e muito mais. Consulte o Capítulo 1 - Mailing Lists para mais informações.

- The Linux Documentation Project O objetivo do The Linux Documentation Project (TLDP) é colaborar em todas as questões da documentação Linux. O TLDP apresenta uma grande coleção de HOWTOs, guias e man pages. Ele está localizado em https://tldp.org/.
