# ii. Público-alvo

Existem muitas razões pelas quais você gostaria de ler este livro. Uma das perguntas que muitas pessoas fazem é: “por que passar por todo o trabalho de construir manualmente um sistema Linux do zero quando você pode simplesmente baixar e instalar um já existente?”

Uma razão importante para a existência deste projeto é ajudar você a aprender como um sistema Linux funciona de dentro para fora. Construir um sistema LFS ajuda a demonstrar o que faz o Linux funcionar e como as coisas trabalham juntas e dependem umas das outras. Uma das melhores coisas que esta experiência de aprendizado pode proporcionar é a capacidade de personalizar um sistema Linux para atender às suas próprias necessidades exclusivas.

Outro benefício chave do LFS é que ele lhe dá controle do sistema sem depender da implementação Linux de outra pessoa. Com o LFS, você está no controle. Você dita cada aspecto do seu sistema.

O LFS permite criar sistemas Linux muito compactos. Com outras distribuições, você é frequentemente forçado a instalar muitos programas que você nem usa nem entende. Esses programas desperdiçam recursos. Você pode argumentar que, com os discos rígidos e CPUs de hoje, recursos desperdiçados não são mais uma consideração. Às vezes, no entanto, você ainda é limitado pelo tamanho do sistema, se nada mais. Pense em CDs bootáveis, pendrives USB e sistemas embarcados. Essas são áreas onde o LFS pode ser benéfico.

Outra vantagem de um sistema Linux construído sob medida é a segurança. Ao compilar o sistema inteiro a partir do código-fonte, você tem o poder de auditar tudo e aplicar todos os patches de segurança que desejar. Você não precisa esperar que outra pessoa compile pacotes binários que corrijam uma falha de segurança. A menos que você examine o patch e o implemente você mesmo, você não tem garantia de que o novo pacote binário foi construído corretamente e corrige o problema de forma adequada.

O objetivo do Linux From Scratch é construir um sistema de nível fundamental completo e utilizável. Se você não deseja construir seu próprio sistema Linux do zero, você pode, no entanto, se beneficiar das informações contidas neste livro.

Existem muitas boas razões para construir seu próprio sistema LFS para listá-las todas aqui. No final, a educação é de longe a razão mais importante. À medida que você continua sua experiência com o LFS, você descobrirá o poder que a informação e o conhecimento podem trazer.
