# ix. Erratas e Avisos de Segurança

O software usado para criar um sistema LFS está sendo constantemente atualizado e aprimorado. Alertas de segurança e correções de bugs podem se tornar disponíveis após o lançamento do livro LFS. Para verificar se as versões dos packages ou as instruções nesta versão do LFS precisam de alguma modificação — para reparar vulnerabilidades de segurança ou corrigir outros bugs — por favor, visite https://www.linuxfromscratch.org/lfs/errata/12.4-systemd/ antes de prosseguir com seu build. Você deve anotar quaisquer mudanças mostradas e aplicá-las às seções relevantes do livro enquanto constrói o sistema LFS.

Além disso, os editores do Linux From Scratch mantêm uma lista de vulnerabilidades de segurança descobertas após o lançamento de um livro. Para ler a lista, por favor, visite https://www.linuxfromscratch.org/lfs/advisories/ antes de prosseguir com seu build. Você deve aplicar as mudanças sugeridas pelos advisories às seções relevantes do livro enquanto constrói o sistema LFS. E, se você for usar o sistema LFS como um sistema de desktop ou servidor real, você deve continuar a consultar os advisories e corrigir quaisquer vulnerabilidades de segurança, mesmo quando o sistema LFS tiver sido completamente construído.
