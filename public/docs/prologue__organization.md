# viii. Estrutura

Este livro está dividido nas seguintes partes.

## Parte I - Introdução

A Parte I explica algumas notas importantes sobre como prosseguir com a instalação do LFS. Esta seção também fornece meta-informações sobre o livro.

## Parte II - Preparando para a Build

A Parte II descreve como se preparar para o processo de build — criando uma partição, baixando os packages e compilando ferramentas temporárias.

## Parte III - Construindo a LFS Cross Toolchain e Ferramentas Temporárias

A Parte III fornece instruções para construir as ferramentas necessárias para a construção do sistema LFS final.

## Parte IV - Construindo o Sistema LFS

A Parte IV guia o leitor através da construção do sistema LFS — compilando e instalando todos os packages um por um, configurando os boot scripts e instalando o kernel. O sistema Linux resultante é a base sobre a qual outro software pode ser construído para expandir o sistema conforme desejado. No final deste livro, há uma referência fácil de usar listando todos os programas, bibliotecas e arquivos importantes que foram instalados.

## Parte V - Apêndices

A Parte V fornece informações sobre o próprio livro, incluindo acrônimos e termos, agradecimentos, dependências de package, uma listagem dos boot scripts do LFS, licenças para a distribuição do livro e um índice abrangente de packages, programas, bibliotecas e scripts.
