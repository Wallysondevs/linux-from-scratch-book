# iv. Pré-requisitos

Construir um sistema LFS não é uma tarefa simples. Requer um certo nível de conhecimento existente em administração de sistemas Unix a fim de resolver problemas e executar corretamente os comandos listados. Em particular, como um mínimo absoluto, você já deve saber como usar a linha de comando (shell) para copiar ou mover arquivos e diretórios, listar o conteúdo de diretórios e arquivos, e mudar o diretório atual. Também é esperado que você saiba como usar e instalar software Linux.

Como o livro LFS assume pelo menos este nível básico de habilidade, os vários fóruns de suporte LFS provavelmente não lhe fornecerão muita assistência nessas áreas. Você descobrirá que suas perguntas sobre tal conhecimento básico provavelmente ficarão sem resposta (ou você será simplesmente encaminhado para a lista de leitura prévia essencial do LFS).

Antes de construir um sistema LFS, recomendamos que você leia estes artigos:

- Software-Building-HOWTO https://tldp.org/HOWTO/Software-Building-HOWTO.html Este é um guia abrangente para construir e instalar pacotes de software Unix “genéricos” sob Linux. Embora tenha sido escrito há algum tempo, ainda fornece um bom resumo das técnicas básicas usadas para construir e instalar software.

- Beginner's Guide to Installing from Source https://moi.vonos.net/linux/beginners-installing-from-source/ Este guia fornece um bom resumo das habilidades e técnicas básicas necessárias para construir software a partir do código-fonte.
